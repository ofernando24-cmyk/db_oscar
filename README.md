# Lumi Commerce Lakehouse

Proyecto práctico de capacitación Databricks.

## Casos del curso

1. **Lumi Commerce Lakehouse**
   - Lakehouse
   - Arquitectura Medallion
   - Delta Lake
   - SQL
   - KPIs
   - Dashboards

2. **Lluvia, Caña y Bagazo**
   - Caso espejo empresarial
   - EDA
   - Correlación
   - Feature engineering
   - MLflow
   - Batch inference
   - MLOps

## Principios del proyecto

- Código versionado.
- Notebooks organizados.
- Separación entre datos, código, SQL, documentación y entregables.
- Enfoque educativo en Databricks Free Edition.
- Explicación de escalamiento a Azure Databricks empresarial.
